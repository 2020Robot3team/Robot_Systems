# Robot_Systems
ロボットシステム学の課題で作成したプログラムです。
LEDの点滅の時間を調整しモールス信号として伝えます。
# 環境
- Rasberry pi ModelB
- Ubuntu 18.04LTS
- LED
- ブレッドボード
- 抵抗　
# 回路
Rasberry pi の22pinに＋端子　　
　　35pinに－端子を指す

# 実行手順
<https://github.com/2020Robot3team/Robot_Systems>より
コードをコピーし
'git clone'でぺーストしインストールしてください。
